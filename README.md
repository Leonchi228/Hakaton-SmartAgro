**train.py** содержит логику обучения нейросетевых моделей и процедуру сохранения весов для каждой сельскохозяйственной культуры отдельно.
**conversion.py** отвечает за оптимизацию и конвертацию обученных моделей из формата PyTorch  в формат TorchScript (.ptl), специально предназначенный для эффективной работы на мобильных устройствах.
**mobile.app** содержит исходный код мобильного приложения (написанного на Flutter), которое использует сконвертированные модели для классификации болезней растений в реальном времени.
